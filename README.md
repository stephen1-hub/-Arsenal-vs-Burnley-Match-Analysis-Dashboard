# ⚽ Arsenal vs Burnley — Match Analysis Dashboard

An interactive football analytics dashboard built to analyse the match between Arsenal and Burnley using Expected Goals (xG), shot maps, shot outcomes, and tactical performance metrics.

This project transforms raw event-level football data into tactical insights and visual storytelling.

---

# 📌 Project Objective

The goal of this project was to:

* Analyse attacking and defensive performance
* Measure chance quality using Expected Goals (xG)
* Compare open-play vs set-piece efficiency
* Identify key tactical patterns
* Visualise player and team performance interactively

This dashboard was designed to simulate how a modern football data analyst communicates insights to coaches, analysts, recruiters, or fans.

---

# 📊 Key Insights

## Arsenal Dominated Chance Quality

* Arsenal generated significantly higher xG than Burnley
* Sustained territorial pressure led to repeated attacks in dangerous areas

## Set Pieces Decided the Match

* Corner situations generated the highest-quality chances
* Kai Havertz scored from the highest xG chance of the match

## Burnley’s Low Block Reduced Open-Play Efficiency

* Arsenal controlled possession but many shots were blocked or saved
* Burnley defended compactly in open play but struggled aerially

## Shot Maps Revealed Territorial Control

* Arsenal occupied central and half-space shooting zones
* Burnley’s attempts came mostly from low-value wide areas

# 🧠 Tactical Analysis Summary

The data suggests Arsenal’s dominance came through:

* Sustained territorial pressure
* Half-space occupation
* Efficient set-piece execution
* Superior box presence

Meanwhile, Burnley:

* defended compactly,
* reduced some open-play penetration,
* but struggled against aerial and set-piece situations.
